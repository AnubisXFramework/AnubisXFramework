**Publication-Ready Table Specifications**

*Classification: PUBLIC (C0)*

This directory documents tables used by the AnubisX Framework foundational paper. Each entry describes the purpose, data source, and manuscript section reference.

---

## Overview

The framework paper includes tables covering comparative analysis of approaches, axiomatic system summaries, mathematical framework layer overviews, likelihood ratio interpretation scales, architecture summaries, behavioral modality descriptions, experimental result distributions, performance metrics, feature distributions, validation results, claims validation, and prototype limitation documentation.

## Data Source Reference

All tables derive from the framework's formal specifications, experimental results, and author analysis as documented in the manuscript.
