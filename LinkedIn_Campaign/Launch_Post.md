**Headline: Introducing AnubisX Framework — Open-Source Behavioral Identity Attribution**

I'm pleased to announce the Foundational Release of the AnubisX Framework, an open-source behavioral identity attribution system.

**What is AnubisX?**

AnubisX is a research framework that extracts behavioral identity fingerprints from text and enables similarity-based user attribution. It combines stylometric analysis — the measurement of an individual's unique writing patterns — with similarity search to answer the question: "Who wrote this text?"

The framework is designed as a modular, multi-layered system supporting multiple analytical modalities: stylometric fingerprinting, temporal pattern analysis, social graph construction, fingerprint stability tracking, and cross-platform adaptation.

**What's in this release?**

This release delivers the complete theoretical specification alongside a prototype that implements the stylometric modality end-to-end. Key experimental results include:

- Stylometric fingerprints successfully extracted from user accounts
- Similarity search demonstrating high precision
- Cross-user differentiation demonstrated — mean similarity between different users is significantly lower than self-similarity
- Linguistic analysis validated across the dataset

**What we're honest about**

This release is a foundation, not a finished product. We are transparent about current limitations: some modalities are blocked by data access issues, temporal and graph analysis layers require additional data collection, and the prototype remains single-platform with limited computational scope. All limitations are documented in the release package.

**Why open-source?**

Behavioral identity attribution has applications in cybersecurity, disinformation research, forensic investigation, and social science. By releasing this framework openly under CC BY 4.0, we hope to enable collaboration across academia, industry, and independent research — and to subject our methods to the scrutiny of peer review.

**Get involved**

The full release is available on GitHub. We are actively seeking collaborators for:

- Multi-platform data collection and adapter implementation
- Large-scale evaluation with labeled ground truth data
- Temporal and graph analysis module development
- Peer review and methodological critique

Whether you're a computational social scientist, a forensic analyst, a machine learning researcher, or a developer working on attribution — your contributions are welcome.
