| Term | Definition |
|---|---|
| **Attribution** | The process of determining the identity of the human operator behind observed digital behavior |
| **Behavioral Profile** | A structured representation of an individual's observed behavioral patterns within a specific modality |
| **Behavioral Signature** | A recurring pattern of human-machine interaction that is individual-specific and persistent |
| **Chrono-Profiling** | The analysis of temporal behavior patterns for identity attribution |
| **Cognitive Centroid** | The mathematical attractor of an individual's behavioral patterns in multi-dimensional feature space |
| **Counter-Forensics** | Deliberate modification of behavior to evade attribution |
| **Daubert Standard** | Legal standard for admissibility of scientific evidence in US federal courts |
| **Dempster-Shafer Theory** | Mathematical framework for reasoning under uncertainty by combining evidence from multiple sources |
| **Environmental Media Forensics** | Analysis of digital environment artifacts for behavioral profiling |
| **Evidence Weighting** | The process of adjusting evidence strength based on quality, relevance, and recency |
| **Feature Extraction** | The process of transforming raw behavioral data into structured feature vectors |
| **Forensic Stylometry** | The analysis of linguistic patterns for identity attribution |
| **Fusion** | The combination of evidence from multiple modalities into a single attribution decision |
| **Identity Intelligence (IdINT)** | The proposed formal forensic discipline for behavioral identity attribution |
| **Lexical Features** | Text-based features including vocabulary richness, word length distributions, punctuation usage, and character-level patterns |
| **Likelihood Ratio (LR)** | A statistical measure of evidence strength comparing the probability of evidence under competing hypotheses |
| **Multi-Modal Convergence** | The principle that multiple independent behavioral signals converge on the same identity |
| **Relational Network Analysis** | The analysis of social graph structure and communication patterns for attribution |
| **Stylometric Fingerprint** | A feature vector encoding an individual's linguistic patterns |
| **Terminal Execution Profiling** | The analysis of human-machine interaction patterns for attribution |
| **Type-Token Ratio (TTR)** | A measure of lexical diversity calculated as the ratio of unique words (types) to total words (tokens) |
